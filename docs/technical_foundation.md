# Technical Foundation

This project is scaffolded with React 19, Tailwind CSS v4, Vite 7, and TypeScript strict mode.

- React 19 is the current stable default in April 2026 and is compatible with the module specs.
- Tailwind v4 is used with the `@tailwindcss/vite` plugin and CSS-first config in `src/styles/globals.css`.
- The app uses Supabase for auth, storage, realtime, and edge functions.
- Client state uses TanStack Query and Zustand, forms use React Hook Form, validation uses Zod.
